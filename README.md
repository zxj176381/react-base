# React 项目

## 使用的第三方包

```
react
typescript
react-router-dom
zustand
react-query
react-use
dayjs
tailwindcss/styled-components
ui组件库另外选择 antD mui 等
```

## 目录结构

```
router -- 路由
services -- 服务器接口
services/config -- ajax 封装
store 状态管理
types 全局ts类型
utils 公用工具函数和业务函数
```
